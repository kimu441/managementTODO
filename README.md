# 📝 本格ローカル完結型TODOアプリ

このアプリは、外部へのデータ送信を一切行わず、100%ローカル環境（あなたのMac内）だけで完結する、安全・高速・超軽量な個人専用のTODO管理アプリです。

## 🛡️ セキュリティと安全性の公約
- **完全ローカル保存：** タスクの内容やデータは、ブラウザの `LocalStorage` にのみ保存されます。インターネット上のサーバーには1文字も送信されないため、情報漏洩のリスクは物理的にゼロです。
- **依存ライブラリゼロ（ゼロ・マニフェスト）：** サードパーティ製の個人ライブラリやUIキットを一切導入せず、React、TypeScript、素のCSSのみで構築します。これにより、サプライチェーン攻撃（悪意あるコードの混入）を100%封殺します。

## 🚀 主な機能とメリット
1. **平日 / 土日・祝日 タスクの「完全分離タブ」**
   - 「平日の夜にやること（副業リサーチや勉強）」と「土日にやること（メルカリ出品やガッツリ作業）」をタブでパッと切り替え。脳のモード切り替えをスムーズにし、集中力を最大化します。
2. **特定キーワード連動型「AIプロンプト自動生成＆ワンクリックコピー」**
   - タスクに「メルカリ」などのキーワードが入ると、横に `[📋 AIアシスト]` ボタンが出現。押すだけで最適なプロンプトがクリップボードにコピーされ、ChatGPTやGeminiでの作業が爆速化します（外部APIを叩かないため完全無料・安全）。
3. **タスクの「優先度（高・中・低）」と「クイックソート」**
   - タスクが増えても「今、何から手を付けるべきか」を一瞬で視覚化し、迷う時間をゼロにします。
4. **進捗視覚化プログレスバー**
   - 今日の達成度が％で目に見えるため、モチベーションが維持され、タスクを完了させる楽しさを生み出します。

## 🖥️ 配信・起動形態（運用の工数ゼロ）
- **GitHub Pages ＋ PWA（デスクトップアプリ化）：**
  アプリをGitHub Pages（完全無料）に一度デプロイした後は、MacのChrome等から「デスクトップにインストール」するだけで、独立したMacアプリとしてDockやLaunchpadに登録されます。
  2回目以降は、VSCodeも、ターミナルも、ローカルホスト（localhost）の起動も一切不要。アイコンをワンクリックするだけで、一瞬（0.5秒以下）で爆速起動します。

## 🛠️ 技術スタック
- **フロントエンド：** React 19 / TypeScript
- **ビルドツール：** Vite
- **スタイリング：** CSS Modules（素のCSS）
- **データ保存：** Web Storage API (LocalStorage)
