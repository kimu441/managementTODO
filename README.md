# 🚀 TaskBuddy

> **「100% Local & Secure」×「AI自動連携」**
> 使うほどに自分専用のAIアシスタントへと進化する、至高のローカルファースト・タスクバディシステム。

## 🌟 プロジェクトの思想
『TaskBuddy』は、単なる「やることリスト」ではありません。日々のタスク管理（平日／休日／過去ログ）を洗練されたUIで行いながら、**「タスクの文字から最適なAIプロンプトを自動検知して提案する」**という、タスク消化を爆速にするための「AI相棒（バディ）システム」です。

すべてのデータは外部のサーバーではなく、あなたのブラウザ（LocalStorage）にのみ安全に保存されます。

---

## 🚀 唯一無二のコア機能

### 1. 💡 複数ワード対応の「AIプロンプト資産図鑑」
* **OR検索（複数ワード）対応**: キーワードを半角・全角カンマや読点（`,`, `，`, `、`）で区切って複数登録できます（例: `メルカリ,フリマ,出品`）。どれか1つでもタスク名に含まれていれば、AIバディが賢く自動検知します。
* **動的プレースホルダー**: プロンプト文の中に `${todoText}` と書いておくだけで、実行時に現在のタスク名へ自動置換。クリップボードに一発でコピーされます。

### 2. 🔥 NOW FOCUSING（ピン留め集中カード）
* 最優先で取り組むべきタスクを1つだけ「ピン留め」すると、画面上部に**巨大な特設フォーカスカード**として出現。視覚的なノイズを削ぎ落とし、今やるべきことに脳のリソースを100%集中させます。

### 3. ⏳ 過去ログ（2日制限の自動クリーンアップ）
* 完了したタスクは「過去ログ」タブへ自動移動。
* 画面がタスクで埋め尽くされてモチベーションが下がるのを防ぐため、**完了から48時間（2日間）が経過したタスクは裏側で自動的に消去**されるスマートな設計です。

### 4. 📊 成果ジャンル分析 ＆ Markdown日報生成
* 過去2日間に完了したタスクの割合を、ジャンル別（開発・メルカリ・生活・その他）に美しい進捗バーでリアルタイム集計。
* 「成果日報をコピー」ボタンを押すだけで、そのままSlackやNotion、日報に貼り付けられる**綺麗なMarkdown形式の成果報告文章**が自動生成されます。

## 📖 キーワードの複数登録の書き方

プロンプト図鑑で新しいトリガーワードを登録する際は、以下の形式がサポートされています。

* **半角カンマ**: `要約,読書,本,記事`
* **全角カンマ**: `株主優待，優待，投資`
* **読点**: `習慣、ルーティン、継続`

> **挙動仕様**: 上記のように登録した場合、「本を買いに行く」でも「AI本を要約する」でも、指定したワードがどれか1つでも含まれていれば自動的にAIサジェストが発火します。
